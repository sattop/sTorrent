# sTorent: UI/UX спецификация

## Визуальный стиль

`sTorent` должен быть минималистичным, современным и спокойным. Интерфейс не должен выглядеть как старые перегруженные torrent-клиенты. Основной акцент - список загрузок, скорость и управление.

## Темы

- Светлая тема.
- Темная тема.
- Автоматическая тема по системным настройкам.

## Локализация

- В настройках интерфейса должен быть выбор языка.
- Стартовые языки: Русский, English, Español, 中文.
- После смены языка интерфейс должен обновляться сразу или после понятного предложения перезапустить UI.
- Длинные переводы не должны ломать кнопки, таблицы, боковую навигацию и модальные окна.

## Навигация

Основная структура:

- `Загрузки`
- `Очередь`
- `Файлы`
- `Трекеры`
- `Статистика`
- `Настройки`

Для desktop-версии рекомендуется левый сайдбар с иконками и короткими подписями.

## Главный экран

Главный экран показывает:

- список активных и завершенных торрентов;
- строку поиска;
- фильтры статусов;
- общую скорость загрузки и отдачи;
- кнопку добавления торрента;
- кнопку добавления magnet-ссылки;
- индикатор состояния сети.
- кнопку или компактный индикатор `Почему медленно?` / `Speed Doctor`.

Карточка или строка торрента должна показывать:

- название;
- прогресс;
- размер;
- скорость загрузки;
- скорость отдачи;
- ETA;
- сиды и пиры;
- статус;
- быстрые действия: пауза, продолжить, открыть папку, удалить.
- диагностический статус: `Оптимально`, `Ограничено лимитом`, `Мало сидов`, `Проблема с портом`, `Трекер не отвечает`, `Диск занят`.

## Экран добавления торрента

После добавления файла или magnet-ссылки пользователь видит:

- название раздачи;
- общий размер;
- список файлов;
- чекбоксы выбора файлов;
- путь сохранения;
- выбор приоритета;
- блок `Умная настройка` с рекомендуемым профилем и объяснением;
- кнопки `Начать`, `Добавить на паузе`, `Отмена`.

## Smart Download Assistant

В диалоге добавления torrent/magnet должен быть блок умной настройки:

- рекомендуемый профиль;
- краткая причина рекомендации;
- предупреждения;
- действия, которые будут применены;
- кнопка `Принять`;
- кнопка `Изменить`;
- режим `Ручная настройка`.

Профили:

- `Максимальная скорость`
- `Смотреть во время загрузки`
- `Ночной режим`
- `Приватный трекер`
- `VPN-интерфейс`
- `Экономия трафика`
- `Ручной режим`

## Speed Doctor

Speed Doctor должен показывать:

- главный диагноз;
- 2-4 причины низкой скорости;
- быстрые безопасные действия;
- кнопку `Показать подробности`;
- кнопку `Скопировать отчет`.

Пример текста:

```text
Скорость низкая, потому что входящий порт закрыт, а у торрента мало сидов.
```

## Настройки

Разделы настроек:

- `Основные`
- `Загрузки`
- `Сеть`
- `Приватность`
- `Интерфейс`
- `Уведомления`
- `Дополнительно`

### Основные

- папка загрузок по умолчанию;
- запускать приложение вместе с системой;
- сворачивать в трей;
- спрашивать подтверждение перед удалением данных.

### Загрузки

- максимальное количество активных загрузок;
- максимальное количество активных раздач;
- правило сидирования по времени;
- правило сидирования по ratio;
- проверять файлы после завершения.

### Сеть

- входящий порт;
- случайный порт при запуске;
- UPnP/NAT-PMP;
- лимиты скорости загрузки и отдачи;
- расписание лимитов;
- лимит подключений на торрент;
- общий лимит подключений;
- выбор сетевого интерфейса.

### Приватность

- включить/выключить DHT;
- включить/выключить PEX;
- включить/выключить LSD;
- режим приватных трекеров;
- настройки прокси;
- отключение телеметрии.

## Пустые состояния

Если торрентов нет, главный экран показывает простое состояние:

- крупная кнопка добавления `.torrent`;
- поле для magnet-ссылки;
- короткий текст о том, что список загрузок пуст.

## Уведомления

Приложение должно показывать уведомления:

- загрузка завершена;
- ошибка доступа к файлу;
- нет места на диске;
- потеряно соединение с трекером;
- порт недоступен;
- требуется подтверждение удаления данных.

## Поведение интерфейса

- Все основные действия должны быть доступны за 1-2 клика.
- Опасные действия требуют подтверждения.
- Долгие операции показывают прогресс.
- Пользователь должен понимать, активна ли загрузка, стоит ли она в очереди или заблокирована ошибкой.
